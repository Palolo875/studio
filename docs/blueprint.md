# **App Name**: KairuFlow Dashboard

## Core Features:

- Daily Playlist Generation: Generate a daily playlist of tasks based on the user's goals and priorities.
- Personalized Task Recommendations: Recommend tasks tailored to the user's energy levels, intentions, and focus for the day, by using AI to act as a 'tool' and decide when and if to incorporate the time of day, the number of subtasks the task contains, the last time it was addresses, and the user's past completion rate to personalize the recommendations.
- Task List Display: Display the daily playlist tasks in a clear and visually appealing list format.
- Task Completion Tracking: Allow users to mark tasks as complete with a checkbox and track their progress.
- Task Search: Enable users to search for specific tasks or notes within the application.
- Dashboard Overview: Provide a dashboard overview with key information such as the daily playlist, energy level suggestions, and focus recommendations.

## Style Guidelines:

- Primary color: Soft lavender (#E6E6FA) to evoke a sense of calmness and focus.
- Background color: Dark gray (#1A1A1A) to provide a premium and modern feel.
- Accent color: Muted purple (#B0B0D9) for interactive elements and highlights.
- Body and headline font: 'Inter' for a modern and readable experience.
- Use a consistent set of rounded icons for tasks, categories, and actions.
- Maintain a soft and smooth style with rounded corners, subtle shadows, and consistent margins.
- Implement smooth transitions and subtle animations for task completion and playlist updates.
- Primary: Off-white or light gray for backgrounds and main surfaces.
- Secondary: Dark gray or black for text and key elements to ensure readability and contrast.
- Accent: A muted red or coral for highlights, call-to-actions, and important notifications.
